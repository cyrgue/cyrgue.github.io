Method Reference

Method references uses :: notation

System.out::println equivalent to x -> System.out.println(x)
Math::max equivalent to (x, y) -> Math.max(x, y)
String::compareToIgnoreCase equivalent to (x, y) -> x.compareToIgnoreCase(y)
