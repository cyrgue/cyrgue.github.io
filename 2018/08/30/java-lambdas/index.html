java_lambdas

Avant Java 8, il n’existait que deux types de références, des valeurs primitives (char c = ‘C’ ;) et des références vers des objets (String s = new String(“Hello”);)
Depuis Java 8, il est possible d’avoir des références vers des morceaux de codes anonymes (des closures) : les lambdas

Une lambda est la redéfinition d’une méthode d’une interface fonctionnelle sans avoir à faire une classe anonyme, donc gain de ligne de code et de visibilité.
On utilise le nouvel opérateur ->
lambda permet de redéfinir une méthode abstraite d’une interface fonctionnelle donc, par conséquent, elle doit répondre à la signature de la méthode concernée.
